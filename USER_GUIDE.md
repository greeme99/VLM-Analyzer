## 시스템 개요

**VLM MODAPTS 동작분석 시스템**은 제조 공정의 작업 동작을 자동으로 분석하고 표준시간(Standard Time)을 산출하는 웹 기반 애플리케이션입니다.

### 주요 특징
- **자동 분석**: Gemini 2.5 Flash VLM 기술로 영상 자동 분석
- **MODAPTS 표준**: 국제 표준 MODAPTS 코드 자동 부여
- **다크 테마**: 제조 현장 친화적인 다크 테마 대시보드
- **수동 보정**: 분석 결과 수동 보정 및 코멘트 입력
- **다국어 지원**: 한국어/영어 전환 가능
- **보고서 생성**: PDF/Excel 형식 보고서 다운로드

## 주요 기능

### 1. 영상 업로드
제조 공정 영상을 업로드하여 분석을 시작합니다.

**지원 형식**: MP4, AVI, MOV  
**최대 크기**: 500MB  
**권장 길이**: 30초 ~ 5분

**업로드 방법**:
1. "업로드" 메뉴 클릭
2. 파일을 드래그하거나 클릭하여 선택
3. 프로젝트 이름 입력
4. "업로드" 버튼 클릭
5. 분석 완료 대기

### 2. 분석 결과 확인
업로드된 영상이 자동으로 분석되어 MODAPTS 코드가 부여됩니다.

**결과 화면 구성**:
- **요약**: 총 MOD, 표준시간, 동작 수, 평균 신뢰도
- **차트**: MOD 분포, 코드 유형 분포
- **상세 목록**: 각 동작별 코드, MOD, 시간, 신뢰도
- **보정 이력**: 수정된 코드 기록

### 3. 수동 보정
분석 결과가 부정확한 경우 수동으로 보정할 수 있습니다.

**보정 방법**:
1. 상세 목록에서 수정할 동작의 "편집" 버튼 클릭
2. 새로운 MODAPTS 코드 선택
3. 보정 사유 입력 (선택사항)
4. 추가 코멘트 입력 (선택사항)
5. "보정 저장" 클릭

**일괄 보정**:
- 여러 동작을 한 번에 같은 코드로 변경 가능
- "일괄 보정" 패널에서 동작 선택 후 코드 적용

### 4. 보고서 생성
분석 결과를 PDF 또는 Excel 형식으로 다운로드합니다.

**보고서 포함 내용**:
- 프로젝트 정보 및 분석 요약
- 각 동작별 MODAPTS 코드, MOD, 시간
- 코드 유형 분포 통계
- 보정 이력 기록

**다운로드 방법**:
1. 프로젝트 상세 페이지에서 "보고서" 버튼 클릭
2. 원하는 형식 선택 (PDF/Excel)
3. 다운로드 시작

### 5. 분석 이력 관리
과거 분석 프로젝트를 조회하고 비교할 수 있습니다.

**프로젝트 목록**:
- 모든 분석 프로젝트 조회
- 프로젝트 검색
- 상태별 필터링 (완료/분석 중)
- 프로젝트 상세 보기

## 사용 방법

### 기본 워크플로우

```
1. 영상 준비
   ↓
2. 시스템 접속 및 로그인
   ↓
3. 영상 업로드
   ↓
4. 자동 분석 대기 (1-5분)
   ↓
5. 분석 결과 확인
   ↓
6. 필요시 수동 보정
   ↓
7. 보고서 생성 및 다운로드
```

### 언어 변경
1. 우측 상단의 언어 선택 드롭다운 클릭
2. "한국어" 또는 "English" 선택
3. 인터페이스가 자동으로 변경됨

### 프로젝트 삭제
1. 프로젝트 목록에서 삭제할 프로젝트 선택
2. "삭제" 버튼 클릭
3. 확인 대화상자에서 "삭제" 클릭

## MODAPTS 코드 참고

### 이동 (Move) - M1~M6
| 코드 | 설명 | 거리 | MOD 범위 |
|------|------|------|---------|
| M1 | 손가락 이동 | 2.5cm | 2 |
| M2 | 손 이동 | 5cm | 2 |
| M3 | 팔뚝 이동 | 15cm | 3 |
| M4 | 팔 이동 | 30cm | 4 |
| M5 | 팔 + 몸 이동 | 60cm | 5 |
| M6 | 전신 이동 | 90cm+ | 6 |

### 집기 (Get) - G0~G3
| 코드 | 설명 | MOD |
|------|------|-----|
| G0 | 쉬운 집기 (부품이 명확함) | 2 |
| G1 | 일반 집기 | 3 |
| G2 | 어려운 집기 (부품이 불명확) | 4 |
| G3 | 도구 집기 | 5 |

### 놓기 (Place) - P0~P2
| 코드 | 설명 | MOD |
|------|------|-----|
| P0 | 가벼운 놓기 | 2 |
| P1 | 일반 놓기 | 3 |
| P2 | 조심스러운 놓기 (정확도 필요) | 4 |

### 기타 코드
| 코드 | 설명 | MOD |
|------|------|-----|
| D3 | 판단 (결정) | 3 |
| L1 | 읽기 (짧음) | 1 |
| E2 | 눈 이동 | 2 |
| F3 | 발 이동 | 3 |
| R2 | 읽기 (일반) | 2 |

### 시간 계산
- **1 MOD = 0.129초**
- 예: M3 (3 MOD) = 3 × 0.129 = 0.387초

## FAQ

### Q: 분석이 얼마나 걸리나요?
**A**: 일반적으로 1-5분 소요됩니다. 영상 길이와 복잡도에 따라 달라질 수 있습니다.

### Q: 분석 정확도는 어느 정도인가요?
**A**: 평균 85% 이상의 신뢰도로 분석됩니다. 신뢰도가 낮은 부분은 수동 보정을 권장합니다.

### Q: 어떤 영상 형식을 지원하나요?
**A**: MP4, AVI, MOV 형식을 지원합니다. 최대 500MB까지 업로드 가능합니다.

### Q: 보정한 내용이 저장되나요?
**A**: 네, 모든 보정 내용은 자동으로 저장되며 보정 이력에서 확인할 수 있습니다.

### Q: 보고서를 여러 번 다운로드할 수 있나요?
**A**: 네, 언제든지 프로젝트 상세 페이지에서 보고서를 다시 다운로드할 수 있습니다.

### Q: 분석 결과를 다른 사람과 공유할 수 있나요?
**A**: 현재는 보고서 다운로드를 통해 공유 가능합니다. 향후 공유 기능이 추가될 예정입니다.

### Q: 언어를 변경해도 저장된 데이터는 유지되나요?
**A**: 네, 언어 변경은 UI 표시만 변경하며 데이터는 영향을 받지 않습니다.

### Q: 시스템이 지원하는 최대 영상 길이는?
**A**: 기술적으로는 제한이 없지만, 분석 정확도와 처리 시간을 고려하여 5분 이내를 권장합니다.
